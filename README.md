# ☕︎ Projeto Gerador de Senha

**Gerador de senha** desenvolvido em **HTML**, **CSS** e **JavaScript**, que permite a criação de senhas personalizáveis de acordo com o tamanho definido pelo usuário, inspirado no tutorial do Youtube [Criando projeto completo com HTML, CSS e JavaScript](https://youtu.be/i6t2jaRxos4?si=CjixHi2UeGTF5G1q).

## ☕︎ Funcionalidades
- Geração de senha aleatórias.
- Escolha do tamanho da senha através de um controle deslizante.
- Copiar a senha gerada.
- Interface simples e intuitiva

## ☕︎ Tecnologias Utilizadas
- **HTML5** - Estruturação semântica e responsiva.
- **CSS3** - Estilização moderna com foco em design limpo e agradável.
- **JavaSc** - Recursos modernos da linguagem como let / const, funções de seta e manipulação dinâmica do DOM.

## ☕︎ Objetivo do Projeto
Criar uma ferramenta simples e intuitiva que permita ao usuário gerar senhas fortes e seguras de maneira rápida, personalizando o tamanho e garantindo facilidade de uso.
